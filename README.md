##**技术框架**

- spring boot,spring mvc,spring：spring cloud暂时未集成
- spring security：安全及权限框架
- spring session：session共享
- thymeleaf：模板引擎
- oauth2：单点登陆
- cas：单点登陆
- quartz：任务
- mysql：数据库
- mybatis：持久层访问
- mybatisplus：持久层访问增强
- ehcached：本地磁盘缓存
- redis：内存与磁盘型缓存，用于spring session持久化session
- memcached：内存缓存
- mongodb：nosql数据库
- log4j2 + ELK：日志处理框架
